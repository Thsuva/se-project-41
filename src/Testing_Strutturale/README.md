# se-project-41 TestStrutturale

nella cartella vengono presentati gli scripit usati per il testing strutturale

i test vengono svolti dallo script testRunner.m ma sono ubicati in test.m

constando di 4 funzioni che si prefiggono di testare le uniche 4 funzioni per cui si è riuscito a replicare un output costante le funzioni in oggetto sono

splitCrossVal che controlla che il dataset sia diviso in due porzioni casuali: Test e Train set (70 30)

errorChecker che controlla il tipo di errore generato da ogniuno dei file presenti nella cartella corrisponda a quello corretto

errorHandler che controlla sia che il ritorno sia corretto sia che la stringa di errore fornita all'utente sia corretta rispetto all'errore rilevato

deleteUselessFeature che controlla che vengano eliminate le corrette feature il cui valore di imp è 0

sono state ignorate le funzioni di lettura e scrittura che sono state esaustivamente testate durante il testing funzionale
